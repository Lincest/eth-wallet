## Eth wallet

一个以太坊钱包。

## tech stack

### 前端

- `angular`: [Angular](https://angular.cn/) `version: @angular/cli@13.2.5`
- `primeng`: [PrimeNG (primefaces.org)](https://www.primefaces.org/primeng/#/theming) `version: 13.2.0`

### 后端

框架: `gin`

## file structure

`./doc/*`: 开发过程的记录文档

`./src/main/front-end`: 项目前端

`./src/main/back-end`: 项目后端
